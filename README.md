# 🌐 Personal Portfolio Website

A personal portfolio website built with **HTML & CSS only** as part of the **Synent Internship — Task 1**.

---

## 📸 Preview

> A clean, dark-themed single-page portfolio showcasing my profile, skills, and contact information.

---

## 📋 Sections

| Section | Description |
|---|---|
| **Header** | Name, title, and a call-to-action button |
| **About Me** | Short bio and personal fact cards |
| **Skills** | HTML, CSS, JS, React, Node.js, PostgreSQL |
| **Contact** | Email, GitHub, LinkedIn, and a message form |

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Google Fonts (Space Grotesk & Space Mono)

---

## ✨ Features

- Dark theme with purple accent color
- Fully responsive (mobile, tablet, desktop)
- Fixed navigation with smooth scroll
- Skill cards with progress bars
- Clean, developer-focused design

---

## 🚀 Getting Started

No installation needed. Just open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/DuDu21cs/synent-task1-personal-portfolio-DuresaChemeda.git

# Open in browser
cd synent-task1-personal-portfolio-DuresaChemeda
open index.html
```
